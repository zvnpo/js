# TypeScript
= JS 문법 + 추가 기능  

: 선택적 정적 타입 언어/도구 (설정을 통해 컴파일 시점에 타입 체크)  
: TS 컴파일러를 통해 JS로 컴파일되는 컴파일 언어 (JS 엔진이 TS를 실행하지 못함)  


**구성 요소**  
- TypeScript Language
- TypeScript Compiler (TSC)
- TypeScript Language Service


**install**
```bash
npm install -g typescript
yarn global add typescript

tsc -v
```


**tsconfig.json**  
: TS 컴파일러를 위한 설정 파일
